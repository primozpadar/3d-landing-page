### Tools used

- [THREE.JS](https://threejs.org/)
- [dot.gui](https://www.npmjs.com/package/dat.gui)

### Development

```bash
# Development
yarn dev

# Build
yarn build
# or
npm run build
```

A few parameters can be modified through dot.gui interface
